天下苦Classin久矣！！！

2022-10-30

有谁会喜欢上网课啊！！！

自Classin某版本开始起，Classin新增了“专注学习模式”，这个设置是机构后台搞的，老师和学生都不能自主取消。

“专注学习模式”会让教室窗口强制全屏且置顶，后台开啥都显示不出来啊！！！

如何破解“专注学习模式”

非通用方法

Windows 8/8.1

  • Windows 8/8.1的Modern应用并不受桌面应用程序的限制（但是Modern应用能干啥捏。。。好像最大的作用是上IE？）

Windows 10

  • 按下 Windows+Tab（对于Mac装Windows的用command+Tab）可打开时间线/多桌面，点击最上面一行的加号新建桌面。
  • 要切的时候可以Win+Tab再调出，或者用Ctrl（Mac-control）+Win（Mac-command）+左右键（← →）快速切换。

Windows 11

  • Windows 10的方法照样有效。
  • 不过如果宁有办法搞上安卓子系统，可以搞搞Classin的安卓版本。

Linux

  • Deepin应用商店有Wine封装的Classin，亲测勉强可以使用。
  • 不过可以装个Anbox（如果宁用Deepin或者UOS，就已经自带UEngine，但是要提前安装好UEngine运行器）运行安卓版本

通用方法

1.外挂法（我lbw从不开挂doge

  • 打开这个链接
  • 这是一个用C++写的程序，通过读取并修改Classin窗口属性来让专注学习模式失效。

2.分辨率法

  1. 先退出教室
  2. 在宁的控制面板（Windows 7/8.x）或者电脑设置（Windows 8.x）或者设置（Windows 10/11）中调整屏幕分辨率，调整到1024x768或者刚好让Classin窗口没法正常显示即可。
  3. 重新进入教室，这时候会提示分辨率不足，别管，直接进。
  4. 这时候全屏无效，先最小化Classin（否则会导致Classin发现你在偷袭，又会恢复的）
  5. 把分辨率调回来，这时候就行了！
